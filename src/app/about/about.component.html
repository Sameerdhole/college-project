Our Project

Our Project consists of developing and improving on the Smart Health care Systems that will help individuals to monitor their vitals such as:

  • Heart Rate
  • Blood Pressure
  • Pulse Oxygen level
  • Temperature

Our group project consists of integrating data from a IOT device ( Node MCU) and uploading the data to the database which is later fetched by the server on request and displayed on the website after the user logs in using the right credentials wwhich is checked by a auth service.The data displayed data can be used to derive infereces regarding the medical conditions of the user and aid them in taking the proper precautions and treatment.Also the User will have the locations of the nearby hospitals based on the gps data recieved from the gps module of the Node MCU.(Application of Neural Networks is in Progress)
